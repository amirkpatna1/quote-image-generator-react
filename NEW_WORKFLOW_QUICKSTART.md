# New 4-Step Workflow - Quick Start Guide

## What's New

Your quote video generator has been completely redesigned with a world-class 4-step workflow:

1. **📥 Media Input** - Add all your ingredients (quotes, videos, audio)
2. **⏱️ Composition & Timing** - Arrange and synchronize everything
3. **🎨 Design & Effects** - Style your quotes and choose transitions
4. **📦 Export** - Review and download your video

## How to Access

The new workflow is the **default experience**. When you open the app, you'll see the new 4-step interface.

### Option A: Use New Workflow (Recommended)
- Just click through the steps
- Follow the intuitive flow
- Download your video

### Option B: Use Classic Workflow
- In Step 1, click "← Use Classic Workflow"
- App switches to the original 3-step interface
- All original features work as before

## Step-by-Step Guide

### Step 1: Media Input (📥)

**Add Your Quotes (Required)**
1. Click "CSV File" to upload quotes
   - Format: CSV with `text` and `author` columns
   - Or use the form to add quotes manually
2. Minimum 1 quote needed to proceed

**Add Videos (Optional)**
1. Check "🎬 Video Clips" checkbox
2. You can upload videos in the next step
3. Max 5 videos, 500MB each

**Add Audio (Optional)**
1. Check "🎵 Audio Track" checkbox
2. Choose one of:
   - **YouTube:** Paste URL (video, Shorts, youtu.be)
   - **Local File:** Upload MP3, WAV, M4A (max 100MB)
3. App shows duration - this sets your video length

**Add Images (Optional)**
1. Check "📸 Background Images" checkbox
2. Choose auto-select from theme or upload custom

**Review Summary**
- See what's ready in the summary box
- ✓ = ready, ○ = pending/not included
- Click "Next" when satisfied

---

### Step 2: Composition & Timing (⏱️)

**Check Duration Status**
- Audio duration vs Content duration displayed
- 🟢 Green = Perfect match
- 🟡 Yellow = Mismatch - need adjustment

**Auto-Sync (If Durations Don't Match)**
1. Click "✨ Auto-Sync to Audio" button
2. App scales everything proportionally
3. Your video will now match audio length

**Adjust Quote Timing**
1. Use slider to set seconds per quote
2. Examples:
   - 2 seconds = short, punchy quotes
   - 4 seconds = average timing
   - 8 seconds = longer reading time
3. See calculation: "10 images × 4s = 40s"

**Upload Videos (If You Checked "Use Videos")**
1. Click to upload video files
2. Drag & drop supported
3. Each shows duration
4. Use ↑↓ buttons to reorder
5. Click ✕ to remove

**See Timeline**
- Visual comparison of audio vs content
- Green bar = audio duration
- Blue bar = content duration
- Helps understand synchronization

**Next Step**
- Click "Next: Design & Effects →"
- Proceed to styling

---

### Step 3: Design & Effects (🎨)

**Choose Quote Styling**

*Theme:*
- Nature (outdoor, green)
- Tech (modern, blue)
- Business (professional, dark)
- Creative (artistic, colorful)
- Minimal (clean, simple)

*Font Style:*
- Sans Serif (modern)
- Serif (classic)
- Monospace (technical)
- Script (elegant)

*Text Color:*
- Click color swatch
- Choose any color
- Drag slider for opacity

*Layout Template:*
- Classic Center (text in middle)
- Top Aligned (text at top)
- Bottom Aligned (text at bottom)
- Minimal (just quote, no frills)

*Overlay Opacity:*
- Adjust how much background shows through
- 0% = fully transparent (see background)
- 100% = fully opaque (solid overlay)

**Choose Slideshow Effect**

8 stunning transition effects:

| Effect | What It Does | Best For |
|--------|-------------|----------|
| ✨ Fade | Smooth fade between quotes | Universal, safe |
| 🔍 Zoom | Zoom in/out on image | Dynamic, engaging |
| ↔️ Pan | Move across the image | Movement, flow |
| 🔄 Rotate | Spin the background | Creative, modern |
| 🎯 Blur-to-Focus | Blur becomes sharp | Attention-grabbing |
| ↕️ Flip | Flip image transition | Unique, stylish |
| ← Slide Left | Slide from right to left | Directional, clear |
| → Slide Right | Slide from left to right | Directional, clear |

**Customize Effect (If Applicable)**

Some effects have parameters:
- **Zoom:** 1.0 = no zoom, 2.0 = double size
- **Pan:** 10 = slow, 50 = fast movement
- **Rotate:** 1° = slight spin, 45° = dramatic spin
- **Blur:** 5 = subtle, 50 = strong blur

**Preview**
- See sample quote with your settings
- Effect name shown at bottom
- Updates in real-time as you change settings

**Next Step**
- Click "Next: Export →"
- Go to final review

---

### Step 4: Export & Review (📦)

**Review Everything**

*Composition Summary:*
- ✓ How many quotes
- ✓ Total quote time
- ✓ How many videos (if added)
- ✓ Video durations
- ✓ Audio track info
- ✓ Final total duration

*Design Settings:*
- Theme chosen
- Text color
- Overlay opacity

*Media Arrangement:*
- 🖼️ Quotes slideshow (purple)
- 🎬 Videos (cyan, if added)
- 🎵 Audio track (pink, if added)

**Choose Export Settings**

*Format:*
- MP4 (H.264) - Universal, compatible everywhere
- WebM (VP8) - Web-optimized, smaller file
- MOV (ProRes) - Professional, high quality

*Quality:*
- Low (360p, 2 Mbps) - Smallest file, fastest
- Medium (720p, 4 Mbps) - Balanced
- High (1080p, 8 Mbps) - Best quality

**See Estimates**
- Estimated file size (e.g., "45 MB")
- Processing time (e.g., "~90 seconds")

**Export Video**
1. Click "✨ Export Video"
2. Progress bar shows %
3. Keep browser open during export
4. File auto-downloads when complete

**Tips**
- Lower quality = faster, smaller file
- High quality = larger file, longer wait
- Use MP4 for social media compatibility
- Use High quality for professional use

## Common Scenarios

### Scenario 1: Create Quick Social Media Video
```
Step 1: Add 5 short quotes, add background music
Step 2: Set 2 seconds per quote (total 10s), no videos
Step 3: Use "Fade" effect, "Tech" theme, white text
Step 4: Export as MP4, Medium quality
Result: ~4MB file, ready for Instagram/TikTok
```

### Scenario 2: Create Motivational Video with Effects
```
Step 1: Add 10 inspiring quotes, YouTube audio (3 min)
Step 2: Auto-sync button does the work
Step 3: Use "Zoom" effect, "Creative" theme, custom color
Step 4: Export as MP4, High quality
Result: ~120MB file, cinematic quality
```

### Scenario 3: Create Video with Custom Clips
```
Step 1: Add 8 quotes, 2 intro/outro videos, local audio file
Step 2: Order videos (intro → quotes → outro), adjust timing
Step 3: Use "Slide Left" effect, "Minimal" theme
Step 4: Export as MP4, High quality
Result: Professional multi-clip video with audio
```

### Scenario 4: Go Back and Revise
```
In Step 3, realize you want different timing?
Click "← Back" button
Goes back to Step 2, keeps your quotes and audio
Adjust timing, then Next again
```

## Troubleshooting

### "Add quotes first to continue"
- You need at least 1 quote
- Check CSV file has correct format

### Audio not showing duration
- Make sure file is valid MP3/WAV/M4A
- Check file size (max 100MB)
- Try YouTube URL instead

### Video upload stuck
- Check file is valid video format (MP4, MOV, etc.)
- Verify file size under 500MB
- Try smaller file first

### Effects not showing in preview
- Make sure you selected a quote image
- Click on effect to select it
- Refresh page if stuck

### Export taking too long
- Lower quality setting speeds it up
- Close other browser tabs
- Check internet connection

## Tips for Best Results

✨ **Design Tips:**
- Use Fade for professional/corporate
- Use Zoom for dynamic/energetic
- Use Pan for smooth flow
- Use Rotate for creative/modern

⏱️ **Timing Tips:**
- 2-3 seconds = funny/short quotes
- 4-5 seconds = motivational quotes
- 6-8 seconds = longer inspirational text
- Always use Auto-Sync with music

🎵 **Audio Tips:**
- YouTube extracts metadata (no download yet)
- Use local files for reliable audio
- Check duration matches your content
- Test audio quality in export

📱 **Platform Tips:**
- Instagram/TikTok: High quality, 9:16 vertical
- YouTube: High quality, 16:9 widescreen
- Twitter/X: Medium quality, 16:9
- LinkedIn: High quality, 1:1 square
